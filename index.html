Nav. Improvements Under Development

Aug 15, 2020

EQAtlas.ca is undergoing a bunch of backend work to help template-ize our archaeic mapbook. The design will remain intact for the exhilerating nostalgia of classic EQAtlas <3. With the maps templated, headers and pages can be dynamically created by the server so we'll be able to link to connected zones directly from the map pages. This means as you travel from Butcherblock Mountains to Kaladim you want need to go back to the Index page - you'll be able to flow directly to the next map from your current map!

We are going to start with some of the most commonly used zones and expand from there:

  • Greater Faydark
  • Butcherblock Mountains
  • Kaladim (North and South)
  • West Commonlands
  • East Commonlands
  • East Karana
  • West Karana
  • North Karana

- Serynn Dipity
